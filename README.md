# Loguea
Actividad Loguea tu modulo


## 1. Crea un Virtual Enviroment

````shell
python3 -m venv .venv
````

## 2. Activar el Virtual Enviroment

````shell
source .venv/bin/activate
````

## 3. Instalar el archivo Requirements.txt

````shell
pip install -r requirements.txt
````

## 4. Correr el programa

````shell
python manage.py runserver
````

## 5. Desactivar el Virtual Enviroment

````shell
deactivate
````
